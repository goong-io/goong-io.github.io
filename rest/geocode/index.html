Geocoding

On this page

  • Reverse Geocoding
  • Forward Geocoding
  • Get place detail

URL: /Geocode

Method: GET

The Goong Geocoding API does three things:

  • Forward geocoding converts location text into geographic coordinates, turning 91 Trung Knh, Trung Ha, Cu Giy, H Ni into 21.0137443130001,105.798346108
  • Reverse geocoding turns geographic coordinates into place names, turning 21.0137443130001,105.798346108 into 91 Trung Knh, Trung Ha, Cu Giy, H Ni.
  • Get place detail by place_id

Reverse Geocoding

ParameterDescriptionExample
latlngCoordinate string20.981971,105.864323

Example request for reverse Geocoding (coordinates to address)

$ curl "https://rsapi.goong.io/Geocode?latlng=21.013715429594125,%20105.79829597455202&api_key={YOUR_API_KEY}"
